# Troubleshooting

Common issues hit while bringing up a FIBER Lite unit, and their fixes.

- [**SSH Connection Refused**](ssh-connection-refused.md) — `ssh` fails with "Connection
  refused" rather than a password prompt.
- [**SSH Permission Denied**](ssh-permission-denied.md) — SSH is reachable, but the password is
  always rejected.
- [**Docker Compose Plugin Not Found**](docker-compose-plugin-not-found.md) — `apt install
  docker-compose-plugin` fails with "Unable to locate package".
- [**Node-RED Installer 404**](nodered-installer-404.md) — the Node-RED install command 404s
  instead of running.
- [**RTC -EREMOTEIO Error**](rtc-remoteio-error.md) — `dmesg` shows `error -EREMOTEIO: RTC chip
  is not present`.
